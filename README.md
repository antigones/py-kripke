
# py-kripke

A Python project for modeling Kripke frames and evaluating modal logic formulas.

## Features
- Build Kripke frames using NetworkX
- Evaluate logical formulas with SymPy
- Automated testing with pytest

## Requirements
Install dependencies with:
```sh
pip install -r requirements.txt
```

## Usage
Run the main file:
```sh
python main.py
```

Run the tests:
```sh
pytest test.py
```

## Project Structure
- `main.py`: main logic and evaluation functions
- `test.py`: automated tests
- `requirements.txt`: Python dependencies
